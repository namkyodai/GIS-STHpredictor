GIS-STHpredictor
================

This repository contains source code of a plugin for QGIS. The plugin is coded to enable users to do temporal analysis on the Soil Transmitted Dieases (STH) given the historical records. This plugin allows users to generate useful maps in chronological order. This work is financially supported by the World Health Organization (WHO).
